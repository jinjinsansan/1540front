# ウォレット - 入金・反映手順

**最終更新日:** 2025年12月08日

1540BOでは、複数の暗号資産（仮想通貨）を使用した迅速かつ安全な入金が可能です。このページでは、対応しているブロックチェーン、入金アドレスの確認方法、および入金が遅延した場合の対処法について詳しく説明します。

---

## 対応暗号資産とネットワーク

当プラットフォームでは、以下の主要な暗号資産およびネットワーク（ブロックチェーン）に対応しています。入金を行う際は、必ず正しいネットワークを選択してください。誤ったネットワークに送金した場合、資産が失われる可能性があります。

| 暗号資産 | 対応ネットワーク |
| :--- | :--- |
| USDT | ERC20 (Ethereum), TRC20 (TRON) |
| BTC | Bitcoin |
| ETH | ERC20 (Ethereum) |
| TRX | TRC20 (TRON) |
| LTC | Litecoin |
| XRP | Ripple |

## 入金手順

1.  **ウォレットページへ移動:** ログイン後、ダッシュボードから「ウォレット」セクションにアクセスします。
2.  **入金を選択:** 「入金」タブを選び、入金したい暗号資産を選択します。
3.  **ネットワークの選択:** 対応するネットワーク（例: ERC20, TRC20）を選択します。
4.  **入金アドレスの確認:** 表示された専用の入金アドレスをコピーするか、QRコードをスキャンします。このアドレス宛に、お持ちの外部ウォレットまたは取引所から暗号資産を送金してください。

> **重要:** 各暗号資産には、それぞれ専用の入金アドレスが割り当てられています。必ず正しいアドレスに送金してください。

## 確認回数と反映時間

ブロックチェーン上での取引は、複数の「確認（Confirmation）」を経て承認されます。入金がお客様のアカウントに反映されるまでの時間は、ネットワークの混雑状況や必要な確認回数によって異なります。

-   **標準的な反映時間:** 通常、送金後5分から30分程度でアカウントに反映されます。
-   **確認回数:** 各ネットワークで定められた確認回数に達すると、入金が完了します。

## 入金遅延時のトラブルシューティング

入金が30分以上経っても反映されない場合は、以下の点をご確認ください。

1.  **送金ステータスの確認:** ご利用の外部ウォレットや取引所で、送金が正常に完了しているか確認してください。
2.  **トランザクションID（TXID）の確認:** ブロックチェーンエクスプローラー（例: Etherscan, Tronscan）を使用して、トランザクションID（TXID）で取引の状況を追跡します。
3.  **アドレスとネットワークの再確認:** 送金先のアドレスと選択したネットワークが正しかったか、再度確認してください。

上記を確認しても問題が解決しない場合は、以下の情報を添えてサポートチームまでご連絡ください。

-   **トランザクションID（TXID）**
-   **送金した暗号資産の種類と数量**
-   **送金日時**

-   **サポートメール:** support@1540bo.io
