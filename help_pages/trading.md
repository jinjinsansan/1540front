# 取引 - バイナリーオプションの流れ

**最終更新日:** 2025年12月08日

1540BOのバイナリーオプション取引は、シンプルで直感的な操作性が特徴です。このガイドでは、トップゲーム画面の基本的な使い方から、取引チケットの作成、リスク管理までの一連の流れを解説します。

---

## トップゲーム画面の概要

ログイン後、最初に表示されるのが「トップゲーム」画面です。ここには、取引に必要なすべての情報が集約されています。

| 要素 | 説明 |
| :--- | :--- |
| **取引ペア** | BTC/USDT、ETH/USDTなど、取引対象となる暗号資産のペアを選択します。リアルタイムの価格が表示されます。 |
| **価格チャート** | 選択した取引ペアの価格変動を視覚的に確認できます。過去の価格動向を分析し、将来の価格を予測します。 |
| **満期時間** | 取引の判定が行われる時間を選択します。数十秒の超短期から、数時間の長期まで、様々なオプションが用意されています。 |
| **取引履歴** | 過去の取引結果（勝ち/負け）、ベット額、ペイアウト額などを確認できます。 |

## 取引チケットの作成

取引チケットの作成は、わずか3ステップで完了します。

1.  **取引ペアの選択:** 取引したい暗号資産ペアを選びます。
2.  **満期時間の選択:** 取引を終了する時間（満期）を設定します。
3.  **方向の予測とベット:**
    -   価格が上昇すると予測する場合： **「コール（Call）」** ボタンをクリックします。
    -   価格が下落すると予測する場合： **「プット（Put）」** ボタンをクリックします。
    -   ベット額を入力し、取引を確定します。

> **ポイント:** 1540BOでは、世界中のトレーダーのセンチメント（市場心理）をヒートマップで可視化しています。他のトレーダーの動向を参考に、取引戦略を立てることができます。

## リスク管理とペイアウト

バイナリーオプションは、ペイアウト率が固定されているため、リスク管理が容易です。取引を開始する前に、潜在的な利益と損失を正確に把握できます。

-   **ペイアウト率:** 取引が成功した場合に得られる利益の割合です。ペイアウト率は取引ペアや市場の状況によって変動します。
-   **リスク:** 取引が失敗した場合の損失は、ベット額に限定されます。ベット額以上の損失が発生することはありません。

責任ある取引を心掛け、ご自身の資金管理計画に基づいて取引を行ってください。
